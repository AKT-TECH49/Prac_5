\doxysubsection{Detailed Description}
\mbox{\hyperlink{classThermostat}{Thermostat}} class for managing smart home thermostats. 

The \mbox{\hyperlink{classThermostat}{Thermostat}} represents a temp control device in the smart home automation system. It allows monitoring and controlling the temp and managing thermostat states like idle, heating, or cooling.

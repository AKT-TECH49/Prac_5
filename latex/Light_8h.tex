\doxysubsection{Detailed Description}
Defines the \mbox{\hyperlink{classLight}{Light}} class, which represents a smart light in the home automation system. 

This class controls a light, managing its brightness, power, and state through interaction with the \mbox{\hyperlink{classLightState}{Light\+State}} interface.

\begin{DoxyWarning}{Warning}
The class assumes ownership of the \mbox{\hyperlink{classLightState}{Light\+State}} pointer and is responsible for its deletion.
\end{DoxyWarning}
